## Properties

---

### Height

Set height for slider. You need to assign this value before using the create block. Else it will not work.

![Height](./img/height.png)

### Bar Color

Set background color of bar.

![BarColor](./img/bar_color.png)

### Bubble Color

Color of circular _bubble_ inside bar.

![BubbleColor](./img/bubble_color.png)

### Bar Text Color

Color of start and end texts of slider.

![BarTextColor](./img/bar_text_color.png)

### Bubble Text Color

Color of text inside _bubble_.

![BubbleTextColor](./img/bubble_text_color.png)

### Start Text

Start (left) text of slider.

![StartText](./img/start_text.png)

### End Text

End (right) text of slider.

![EndText](./img/end_text.png)

### Font Size

![FontSize](./img/font_size.png)

### Postion

Positon of _bubble_ in range form 0 to 100.

![Postion](./img/position.png)

### Rise Duration

Duration of _bubble_ rise in milliseconds.

![RiseDuration](./img/rise_duration.png)
